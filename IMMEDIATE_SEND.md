## ✅ الحل:

تم تعديل دالة `periodic_sender` لإرسال الآية الأولى **فوراً**، ثم الانتظار 15 دقيقة للآيات التالية.

### الكود القديم:
```python
async def periodic_sender(self):
    while self.running:
        await asyncio.sleep(15 * 60)  # ينتظر 15 دقيقة أولاً
        await self.send_verse_to_channel()
```

### الكود الجديد:
```python
async def periodic_sender(self):
    # إرسال الآية الأولى فوراً
    await self.send_verse_to_channel()
    
    while self.running:
        await asyncio.sleep(15 * 60)  # ثم ينتظر 15 دقيقة
        await self.send_verse_to_channel()
```
