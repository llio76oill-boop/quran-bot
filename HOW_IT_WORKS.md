### 2️⃣ **فئة QuranBot** - المنطق الرئيسي

```python
class QuranBot:
    def __init__(self):
        # إعداد الاتصال بـ API
        self.quran_api = QuranAPI()
        
        # تحميل تفضيلات المستخدمين
        self.user_preferences = self._load_preferences()
        
        # إعداد جدولة المهام
        self.scheduler = BackgroundScheduler()
    
    def schedule_periodic_verses(self):
        # جدولة إرسال آية كل 15 دقيقة
        self.scheduler.add_job(
            func=self.send_verse,
            trigger="interval",
            minutes=15
        )
        self.scheduler.start()
```

**ماذا يفعل؟**
- يدير الاتصال بـ Telegram API
- يدير تفضيلات المستخدمين
- يجدول المهام الدورية
- يعالج الأوامر

### 3️⃣ **مكتبة APScheduler** - الجدولة

```python
from apscheduler.schedulers.background import BackgroundScheduler

scheduler = BackgroundScheduler()

# إضافة مهمة دورية
scheduler.add_job(
    func=send_verse,           # الدالة المراد تنفيذها
    trigger="interval",         # نوع الجدولة
    minutes=15,                 # كل 15 دقيقة
    id="periodic_verse"         # معرف المهمة
)

scheduler.start()  # بدء الجدولة
```

**كيف تعمل؟**
- تعمل في الخلفية (background)
- تتحقق من الوقت باستمرار
- عندما يحين الوقت، تنفذ المهمة
- ثم تنتظر الفترة التالية

## 🛡️ معالجة الأخطاء والاستقرار

### المشكلة 1: انقطاع الإنترنت

```python
# الحل: إعادة المحاولة التلقائية
class RetryStrategy:
    def __init__(self, max_retries=3, backoff=2):
        self.max_retries = max_retries
        self.backoff = backoff
    
    def execute(self, func):
        for attempt in range(self.max_retries):
            try:
                return func()
            except Exception as e:
                if attempt < self.max_retries - 1:
                    wait_time = self.backoff ** attempt
                    time.sleep(wait_time)  # انتظر قبل المحاولة التالية
                else:
                    raise
```

**مثال:**
```
المحاولة 1: فشل (انقطاع إنترنت)
         ↓ انتظر 1 ثانية
المحاولة 2: فشل (الخادم مشغول)
         ↓ انتظر 2 ثانية
المحاولة 3: نجح! ✅
```

### المشكلة 2: فشل متكرر من API

```python
# الحل: Circuit Breaker
class CircuitBreaker:
    def __init__(self, failure_threshold=5, timeout=60):
        self.failure_threshold = failure_threshold
        self.timeout = timeout
        self.failure_count = 0
        self.last_failure_time = None
    
    def can_execute(self):
        if self.failure_count >= self.failure_threshold:
            # الدائرة مفتوحة - لا تحاول
            if time.time() - self.last_failure_time > self.timeout:
                # حاول مرة أخرى بعد انتظار
                self.failure_count = 0
                return True
            return False
        return True
```

**مثال:**
```
الفشل 1: ✗
الفشل 2: ✗
الفشل 3: ✗
الفشل 4: ✗
الفشل 5: ✗ → 🔴 الدائرة مفتوحة! توقف المحاولات
         ↓ انتظر 60 ثانية
         ↓ حاول مرة أخرى
النجاح: ✅ → 🟢 الدائرة مغلقة! استمر
```
