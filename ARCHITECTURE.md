## دورة الإرسال الدوري (Periodic Sending Cycle)

```
┌──────────────────────────────────────────┐
│     البوت يبدأ التشغيل                   │
└──────────────────────────────────────────┘
        │
        ▼

│  • قائمة السور (114)                    │
│  • تفضيلات المستخدمين                   │
│  • السجلات السابقة                      │
└──────────────────────────────────────────┘
        │
        ▼
┌──────────────────────────────────────────┐
│  جدولة المهام:                          │
│  scheduler.add_job(                      │
│    send_verse,                           │
│    trigger="interval",                   │
│    minutes=15                            │
│  )                                       │
└──────────────────────────────────────────┘
        │
        ▼
┌──────────────────────────────────────────┐
│  البوت في حالة الانتظار                 │
│  (يستمع للأوامر والوقت)                 │
└──────────────────────────────────────────┘
        │
        ├─────────────────────────────┐
        │                             │
        ▼                             ▼
    ⏰ حان الوقت!          المستخدم أرسل أمر
        │                             │
        ▼                             ▼
┌──────────────────┐      ┌──────────────────┐
│ send_verse()     │      │ handle_command() │
│ • اختيار آية    │      │ • معالجة الأمر   │
│ • إرسالها        │      │ • الرد           │
└──────────────────┘      └──────────────────┘
        │                             │
        ▼                             ▼
    تسجيل السجل              تسجيل السجل
        │                             │
        └─────────────────┬───────────┘
                          │
                          ▼
                ┌──────────────────────┐
                │ العودة للانتظار     │
                │ (15 دقيقة أخرى)     │
                └──────────────────────┘
```

## الموارد المستخدمة (Resources)

### الذاكرة (Memory)
```
┌─────────────────────────────────┐
│ استهلاك الذاكرة                │
├─────────────────────────────────┤
│ Python Runtime: ~30 MB          │
│ Telegram Bot Library: ~10 MB    │
│ Quran Data (cached): ~5 MB      │
│ User Preferences: ~1 MB         │
│ Scheduler: ~2 MB                │
│ Logs: ~2 MB                     │
├─────────────────────────────────┤
│ الإجمالي: ~50 MB                │
└─────────────────────────────────┘
```

### CPU
```
┌─────────────────────────────────┐
│ استهلاك CPU                    │
├─────────────────────────────────┤
│ في الانتظار: < 1%              │
│ عند الإرسال: 5-10%             │
│ عند البحث: 10-15%              │
└─────────────────────────────────┘
```

### الشبكة (Network)
```
┌─────────────────────────────────┐
│ استهلاك الشبكة                 │
├─────────────────────────────────┤
│ لكل آية مرسلة: ~2 KB           │
│ لكل يوم (96 آية): ~192 KB      │
│ لكل شهر: ~5.76 MB              │
└─────────────────────────────────┘
```

## التوسع المستقبلي (Scalability)

```
النسخة الحالية:
┌─────────────────────────────────┐
│ بوت واحد → مستخدمون محدودون   │
│ ملف JSON → بيانات محدودة       │
│ API واحد → استدعاءات محدودة   │
└─────────────────────────────────┘

النسخة المستقبلية:
┌─────────────────────────────────┐
│ عدة بوتات → آلاف المستخدمين   │
│ قاعدة بيانات → بيانات ضخمة    │
│ عدة APIs → استدعاءات موزعة    │
│ Cache → أداء أفضل              │
│ CDN → توزيع عالمي              │
└─────────────────────────────────┘
```
