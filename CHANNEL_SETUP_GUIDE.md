## 🆕 إنشاء قناة جديدة

### الخطوة 1: افتح تطبيق تلغرام

```
Telegram App
    ↓
اضغط على: ➕ (أيقونة الإنشاء)
```

### الخطوة 2: اختر "قناة جديدة"

```
┌─────────────────────────────────┐
│ ➕ إنشاء                        │
├─────────────────────────────────┤
│ • رسالة جديدة                  │
│ • مجموعة جديدة                 │
│ • قناة جديدة                   │
│ • مجتمع جديد                   │
└─────────────────────────────────┘

اختر: قناة جديدة
```

### الخطوة 3: اختر اسم القناة

```
اكتب: قناة آيات القرآن الكريم

أو: Quran Verses Channel
```

### الخطوة 4: اختر وصف القناة

```
اكتب: 
قناة متخصصة لنشر آيات قصيرة من القرآن الكريم
تُنشر آية كل 15 دقيقة بشكل دوري
```

### الخطوة 5: اختر نوع القناة

```
┌─────────────────────────────────┐
│ نوع القناة                      │
├─────────────────────────────────┤
│ ○ عامة (يمكن للجميع الانضمام)  │
│ ○ خاصة (بدعوة فقط)             │
└─────────────────────────────────┘

اختر: عامة (إذا أردت أن يشترك الجميع)
أو: خاصة (إذا أردت دعوة أشخاص محددين)
```

### ✅ تم إنشاء القناة!

```
القناة الآن جاهزة
معرف القناة: @quran_verses_channel
```

## 🤖 إضافة البوت إلى القناة

### الخطوة 1: افتح القناة

```
اضغط على: القناة التي أنشأتها
```

### الخطوة 2: أضف البوت كمسؤول

```
اضغط على: ⋮ (القائمة الجانبية)
    ↓
اختر: إدارة القناة
    ↓
اختر: المسؤولون
    ↓
اضغط على: ➕ إضافة مسؤول
```

### الخطوة 3: ابحث عن البوت

```
ابحث عن: @quran_verses_bot
(أو اسم البوت الذي أنشأته)

اضغط على: البوت
```

### الخطوة 4: منح الصلاحيات

```
┌─────────────────────────────────┐
│ صلاحيات المسؤول                │
├─────────────────────────────────┤
│ ✓ إدارة القناة                  │
│ ✓ حذف الرسائل                  │
│ ✓ نشر الرسائل                  │
│ ✓ تعديل الرسائل                │
│ ✓ إدارة المشتركين              │
│ ✓ إضافة مسؤولين                │
└─────────────────────────────────┘

تأكد من تفعيل:
✓ نشر الرسائل (مهم جداً!)
```

### ✅ تم إضافة البوت!

```
البوت الآن مسؤول في القناة
ويمكنه نشر الآيات
```

## 🔑 الحصول على معرف القناة

هناك طريقتان للحصول على معرف القناة:

### الطريقة 1: استخدام اسم القناة (الأسهل)

```
معرف القناة: @quran_verses_channel

أضفه في ملف .env:
CHANNEL_ID=@quran_verses_channel
```

### الطريقة 2: استخدام معرف رقمي (الأدق)

#### الخطوة 1: افتح المتصفح

```
اذهب إلى: https://web.telegram.org
```

#### الخطوة 2: افتح القناة

```
اضغط على: القناة الخاصة بك
```

#### الخطوة 3: انسخ المعرف من الرابط

```
الرابط:
https://web.telegram.org/k/#@quran_verses_channel

أو:
https://web.telegram.org/k/#-100123456789

المعرف الرقمي: -100123456789
```

#### الخطوة 3: أضفه في ملف .env

```
CHANNEL_ID=-100123456789
```

```bash

```

```env

# CHANNEL_ID=-100123456789

```

اضغط: Y

## ⚠️ حل المشاكل الشائعة

### المشكلة 1: البوت لا ينشر في القناة

**الحل:**
1. تأكد من أن البوت مسؤول في القناة
2. تأكد من أن معرف القناة صحيح
3. تأكد من أن البوت لديه صلاحية "نشر الرسائل"

---

### المشكلة 2: "Unauthorized" أو "Forbidden"

**الحل:**
1. تأكد من أن البوت مسؤول
2. أعد إضافة البوت كمسؤول
3. تحقق من صلاحياته

---

### المشكلة 3: معرف القناة غير صحيح

**الحل:**
استخدم اسم القناة بدلاً من المعرف الرقمي:
```
CHANNEL_ID=@quran_verses_channel
```

---

### المشكلة 4: البوت يرسل لكن لا تظهر الرسائل

**الحل:**
1. تحقق من السجلات: `tail -f quran_bot.log`
2. تأكد من أن القناة لا تحتاج موافقة على الرسائل
3. تأكد من أن البوت لم يتم حظره
